# 私域直播官网策划文档

适用对象：官网设计、Figma 页面设计、SEO 配置、产品介绍页文案整理

官网名称：私域直播

项目背景：基于现有 `pte-live-im` 与 `pte-live-shop` 相关产品能力，规划一个对外展示的官方网站，用于介绍产品、价格、文档和联系方式。

---

## 1. 官网定位

私域直播是面向私域业务的开源 IM 与直播产品方案，覆盖开源 IM、电商直播、娱乐直播、语聊直播，支持客户独立部署与长期运营。

核心定位：

- 面向有私域业务、直播业务或小程序业务的团队
- 提供开源 IM 与直播产品能力
- 支持独立部署，客户掌握自己的服务器、域名、云资源和数据
- 适合需要二次开发、私有化部署、长期运营的客户

一句话介绍：

> 私域直播提供开源 IM、电商直播、娱乐直播、语聊直播产品方案，支持独立部署，帮助企业快速搭建自己的私域直播系统。

## 4. 产品总览

板块标题：

## 四大核心产品

## 14. FAQ

### 你们提供服务器吗？

不提供。客户需要自行购买服务器和云资源，我们提供产品系统和部署服务。

### 是否包含腾讯云直播流量？

不包含。云直播、云点播、CDN、对象存储等费用由客户自行承担。

### 是否支持微信小程序？

支持接入微信小程序，但小程序账号、认证、发布和微信商户号需要客户自行准备。

### 是否支持私有化部署？

支持。产品可以部署到客户自己的服务器环境中。

### 是否可以二次开发？

可以。开源 IM 和直播产品适合有技术团队的客户做二次开发。
